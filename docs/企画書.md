# Welcome to Party Quiz 特別企画
# チーム対抗クイズ大会 実施企画書

---

## 1. 企画の目的
中学卒業から5年の節目に、当時の思い出と「二十歳の今」の価値観を共有する場を作る。恩師と教え子が同じチームで協力し、独自のシステムを通じた「速度」「度胸」「共感」の競い合いにより、会場全体の一体感を最大化させる。

---

## 2. 開催形式
* **チーム構成**：全8チーム（1チーム約7名）。卒業生55名と先生4名の混合編成とし、先生も一人のプレーヤーとして解答・相談に加わる。
* **操作方法**：各チームに1台の端末（スマホ/タブレット）を配布し、自作の専用システムにアクセス。
* **投影演出**：プロジェクターにて「リアルタイム・ランキング」および「マジョリティ投票集計グラフ」を投影する。

---

## 3. クイズシステム仕様（自作：Next.js / FastAPI / Prisma）
1問目からすべての機能が開放された状態で進行する。

### ① ベッティング（倍率選択）
* 解答前に「x1 / x2 / x3」の倍率を選択する。
* 正解なら得点が選択倍率分加算され、不正解なら同等のポイントが減点されるリスク要素。

### ② 解答速度ボーナス
* 解答が早いチームほど獲得ポイントが高くなる減点タイマー方式を採用。ミリ秒単位の速度判定を行う。

### ③ マジョリティ・ボーナス（価値観同調ゲーム）
* 正解のない設問に対し、会場全体で最も多く選ばれた「多数派（マジョリティ）」と一致した場合にポイントを付与する。
* その選択肢の会場全体での「得票率（％）」をそのままポイント倍率に反映させ、今の空気感を読み切ったチームを評価する。

---

## 4. 景品予算案（総額 42,000円）
システムのログデータを活用し、多角的な視点で表彰を行う。

| 賞の名前 | 選出基準 | 景品内容（案） | 予算配分 |
| :--- | :--- | :--- | :--- |
| **総合優勝** | 最終累計ポイント1位 | 1人3,000円分ギフト券 × 7名 | **21,000円** |
| **総合準優勝** | 最終累計ポイント2位 | 1人1,500円分ギフト券 × 7名 | **10,500円** |
| **不屈の精神賞** | マジョリティ一致率が極端に低い、または特定の苦戦ログから抽出 | 1人1,000円分ギフト品 × 7名 | **7,000円** |
| **ギャンブラー賞** | ベット成功率・最速解答頻度が最も高い個人/チーム | 豪華個人賞（1名分） | **3,500円** |
| **合計** | | | **42,000円** |

---

## 5. 運営・管理
* **所要時間**：約35〜40分（導入、クイズ10〜12問、表彰を含む）。
* **管理者画面**：井上の手元で進行を一括制御（問題切り替え、BGM演出、リアルタイム集計）。
* **デジタル・リザルト**：大会終了後、各チームの戦績データをQRコードで全員に配布し、二次会以降の話題を提供できるよう配慮する。
